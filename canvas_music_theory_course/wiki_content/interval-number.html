Interval Number

Difficulty Level: 3/6
intervalcountinganalysis
Definition: The numeric size of an interval, counting letter names inclusively (unison, 2nd, 3rd, 4th, 5th, 6th, 7th, octave)

Prerequisites

Before studying this concept, make sure you understand:

  • Interval
  • Note Names (Letter Names)

Learning Objectives

By the end of this lesson, you should be able to:

  • Count interval numbers by counting letter names
  • Identify intervals from unison to octave

Examples

  • C to E is a 3rd (C=1, D=2, E=3)
  • F to C is a 5th (F=1, G=2, A=3, B=4, C=5)

Related Concepts

  • Interval Quality
  • Scale Degree

Study Tips

  • Review this concept multiple times until you feel confident
  • Try to create your own examples beyond those provided
  • Connect this concept to music you already know and enjoy
  • Practice identifying this concept in real musical examples
